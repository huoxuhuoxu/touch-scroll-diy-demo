#### touch-scroll-diy模块使用演示

##### use
	
	方式1:
		npm install http-server -g
		进入public目录下执行http-server
		将手机和电脑处于同一wifi下,根据提示,用手机访问: http://xxx.xxx.xx.xx:xx
	方式2:
		项目目录下,执行
		npm install
		npm start
		手机电脑处于同一wifi下,手机访问电脑的ip地址,端口号: 8099, http://xxx.xxx.xx.xx:8099

##### explain
	
	index.html: 
		目前只有基本的滚动容器生成演示
